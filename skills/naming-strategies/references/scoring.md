# Name Scoring Methodology

Every name candidate is scored against 7 weighted criteria. Each criterion receives a score from 1 to 10. The total score is a weighted sum.

## Scoring Criteria

### 1. Memorability (weight: 20%)

How easy is the name to remember after a single exposure?

| Score | Meaning                                         |
| ----- | ----------------------------------------------- |
| 9-10  | Instantly memorable, sticks after one hearing   |
| 7-8   | Remembered after brief exposure, easy to recall |
| 5-6   | Requires a few exposures to stick               |
| 3-4   | Forgettable, blends with other names            |
| 1-2   | Nearly impossible to recall                     |

Factors: brevity, distinctiveness, phonetic rhythm, emotional hook.

### 2. Pronounceability (weight: 15%)

How easy is the name to say correctly across languages?

| Score | Meaning                                                         |
| ----- | --------------------------------------------------------------- |
| 9-10  | Anyone can pronounce it correctly on first try, in any language |
| 7-8   | Clear pronunciation, minor variation across languages           |
| 5-6   | Most people get it right, some hesitation                       |
| 3-4   | Frequently mispronounced, ambiguous stress                      |
| 1-2   | Unpronounceable for most audiences                              |

Factors: phonetic simplicity, consonant clusters, stress patterns, cross-language compatibility.

### 3. Uniqueness (weight: 20%)

How distinct is the name from existing brands in the market?

| Score | Meaning                                      |
| ----- | -------------------------------------------- |
| 9-10  | Completely original, no similar brands exist |
| 7-8   | Highly distinctive, unlikely to be confused  |
| 5-6   | Somewhat unique but echoes existing names    |
| 3-4   | Similar to known brands, risk of confusion   |
| 1-2   | Nearly identical to a major existing brand   |

Factors: phonetic distance from competitors, visual distinctiveness, trademark viability.

### 4. Relevance (weight: 15%)

How well does the name relate to the product, industry, or brand values?

| Score | Meaning                                                              |
| ----- | -------------------------------------------------------------------- |
| 9-10  | Immediately evokes the product/industry without explanation          |
| 7-8   | Clear connection, understood with minimal context                    |
| 5-6   | Loose connection, requires some explanation                          |
| 3-4   | Tenuous link, needs significant brand-building to create association |
| 1-2   | No discernible connection to the product or brand                    |

Factors: semantic fit, industry conventions, audience expectations.

### 5. Emotional Impact (weight: 15%)

What feelings does the name evoke? Do those feelings align with the brand?

| Score | Meaning                                                        |
| ----- | -------------------------------------------------------------- |
| 9-10  | Strong, immediate emotional response aligned with brand values |
| 7-8   | Clear emotional tone, positive associations                    |
| 5-6   | Neutral — neither positive nor negative feelings               |
| 3-4   | Weak or misaligned emotional response                          |
| 1-2   | Negative associations or emotional disconnect                  |

Factors: phonetic mood (hard vs. soft sounds), cultural associations, connotations.

### 6. Scalability (weight: 10%)

Can the brand grow beyond its initial product or market?

| Score | Meaning                                                     |
| ----- | ----------------------------------------------------------- |
| 9-10  | Completely open — could apply to any product line or market |
| 7-8   | Broad enough to expand naturally                            |
| 5-6   | Tied to current product but could stretch with effort       |
| 3-4   | Narrowly defined, expansion would feel forced               |
| 1-2   | Locked to one specific product or feature                   |

Factors: abstraction level, category specificity, geographic limitations.

### 7. Domain-Friendliness (weight: 5%)

How likely is the name to be available as a domain (or close variant)?

| Score | Meaning                                                          |
| ----- | ---------------------------------------------------------------- |
| 9-10  | Short, unique — .com likely available or affordable              |
| 7-8   | Reasonable availability with common TLDs                         |
| 5-6   | Exact .com unlikely but variants exist (.io, .co, prefix/suffix) |
| 3-4   | Common word — domains mostly taken, expensive                    |
| 1-2   | Generic dictionary word — all domains taken                      |

Factors: word length, uniqueness, TLD availability, common misspelling domains.

## Score Calculation

Total score = weighted sum of all criteria:

```
Total = (Memorability x 0.20) + (Pronounceability x 0.15) + (Uniqueness x 0.20)
      + (Relevance x 0.15) + (Emotional Impact x 0.15) + (Scalability x 0.10)
      + (Domain-Friendliness x 0.05)
```

## Thresholds

| Score Range | Rating              | Action                                        |
| ----------- | ------------------- | --------------------------------------------- |
| 8.0+        | Excellent candidate | Advance to final shortlist                    |
| 6.5 - 7.9   | Good candidate      | Include in long list, consider with caveats   |
| 5.0 - 6.4   | Acceptable          | Only include if pool is thin, note weaknesses |
| Below 5.0   | Reject              | Discard, do not present to client             |

## Usage Notes

- Scores are relative to the brief. A name scoring 9 on Relevance for one product may score 3 for another.
- Emotional Impact scoring must reference the brief's target tone and audience.
- Domain-Friendliness has the lowest weight intentionally — a great name with a mediocre domain beats a mediocre name with a perfect domain.
- When comparing names across different strategies, normalize for strategy-inherent biases (e.g., Neologisms naturally score higher on Uniqueness; Compound words naturally score higher on Relevance).
